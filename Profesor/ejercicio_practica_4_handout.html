Ejercicio prctica 4: Optimizacin no lineal con Scipy

Consideremos la funcin de activacin $$ \sigma (x) = \max\left\{x, 0\right\} $$ que, en Machine Learning, es conocida como ReLU (Linear Unit Rectifier). A partir de ella construimos la sencilla red neuronal con dos canales de entrada $(x, y)$ y una sla neurona $$ NN(x, y) = \sigma\left( 2 * x - 3 * y + 1\right) $$ Se pide:

  1. Es diferenciable esta funcin? Por qu?

    No es diferenciable en 2x-3y+1 = 0 porque, cuando 2x-3y+1 > 0, la funcion es lineal y tiene pendiente constante, en cambio, cuando 2x-3y+1 <= 0, la funcion es constante y la derivada es 0.

  2. Define dicha funcin.

# Completar aqu
def relu(x):
    return np.maximum(x, 0)

def NN(v):
    x, y = v
    nn = relu(2 * x - 3 * y + 1)
    return nn
# --------------------

Define ahora la funcin objetivo siguiente: $$ MSE ( x, y) = \frac{1}{2} * \left[ (NN(x,y) -1)^2 + (NN(x,y) - 2)^2\right] $$

In[14]:
# Completar aqu
def MSE(v):
    mse = 1/2 * ((NN(v) - 1)**2 + (NN(v) - 2)**2)
    return mse
# --------------------

Dibuja la funcin $MSE ( x, y)$ en la ventana $-5\leq x\leq 5$, $-5\leq y\leq 5$.

# Completar aqu

initial_guess = r_min + np.random.rand(2) * (r_max - r_min)

result = minimize(MSE, initial_guess, method='powell')

print('Estado : %s' % result['message'])
print('Nmero total de evaluaciones de la funcin objetivo: %d' % result['nfev'])

solution = result['x']
evaluation = MSE(solution)
print('Solucin: f(%s) = %.5f' % (solution, evaluation))
